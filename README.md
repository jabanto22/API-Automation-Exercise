# Pre-requisites

## Python
Please make sure to have the following installed:
* Python 3
* pytest

### How to run the tests
To run the test, go to *tests* directory, then execute `pytest`

## Javascript
Please make sure to have the following installed:
* npm
* mocha
* chai
* supertest
* uuid

### How to run the tests
To run the test, go to *api-test-js* directory, then execute `npm test`
